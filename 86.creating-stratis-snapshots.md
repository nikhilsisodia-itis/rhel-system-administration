# Creating stratis snapshots

## Commands to create, mount, unmount, and delete snapshots

- `stratis fs snapshot <pool_name> <filesystem> <snapshot_name>`: Create a snapshot of a specified filesystem in a given pool.
- `stratis fs snapshot list <pool_name> <filesystem>`: List all snapshots for a specified filesystem in a given pool.
- Create a directory to mount the snapshot:

  ```bash
  mkdir /mnt/snapshot_mount
  ```

- `mount /dev/stratis/<pool_name>/<filesystem>/<snapshot_name> <mountpoint>`: Mount the snapshot on the directory created for mounting the snapshot.
- `stratis fs destroy <pool_name> <filesystem> <snapshot_name>`: This command delete the specified snapshot.
**Note:** You need to unmount a snapshot before deleting it.
- `umount <mount_point>`: Command is used to unmount the filesystem mounted at the specified mount point.
